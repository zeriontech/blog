<p>The <a href="https://developers.zerion.io/introduction" rel="noreferrer">Zerion API documentation</a> has been rebuilt from scratch. The new docs site replaces the previous reference-only format with structured guides, working code examples, and dedicated sections for AI agent builders.</p><p>The docs now start from use cases and walk you through real integrations step by step.</p><h2 id="what-changed">What changed</h2><p>The previous docs were organized around a list of Zerion API endpoints.&nbsp;</p><p>Authentication, REST methods, and a brief "Build with AI" section. It was useful if you already knew what you were looking for, but not much help if you were starting a new project.</p><p>The new docs are structured around several layers:&nbsp;</p><ul><li>A quickstart to get your first request working in minutes,&nbsp;</li><li>A set of recipes for common use cases,&nbsp;</li><li>A core concepts section that covers pagination, rate limits, webhooks, and error handling in dedicated pages,</li><li>Detailed API reference for each endpoint.</li></ul><h2 id="recipes-start-from-the-use-case">Recipes: start from the use case</h2><p>The biggest addition is a recipes section with seven task-focused guides.&nbsp;</p><p>Each recipe combines multiple endpoints to solve a specific problem, with working code in JavaScript and Python.</p><p>The recipes cover the most common integration patterns:</p><ul><li><a href="https://developers.zerion.io/recipes/multi-chain-portfolio" rel="noreferrer"><strong><u>Build a Portfolio Tracker</u></strong></a>. Fetch wallet value, list token holdings, aggregate multiple wallets, and chart performance over time.</li><li><a href="https://developers.zerion.io/recipes/transaction-history" rel="noreferrer"><strong><u>Get a Wallet's Transaction History</u></strong></a>. Parsed, human-readable transactions with filters by type, chain, and date range.</li><li><a href="https://developers.zerion.io/recipes/wallet-activity-alerts" rel="noreferrer"><strong><u>Set Up Wallet Activity Alerts</u></strong></a>. Configure webhooks for real-time notifications without polling.</li><li><a href="https://developers.zerion.io/recipes/wallet-pnl-tracker" rel="noreferrer"><strong><u>Build a Wallet PnL Tracker</u></strong></a>. Realized gains, unrealized gains, cost basis, and per-token performance.</li><li><a href="https://developers.zerion.io/recipes/defi-positions" rel="noreferrer"><strong><u>Get a Wallet's DeFi Positions</u></strong></a>. Lending, staking, and liquidity positions across protocols in one call.</li><li><a href="https://developers.zerion.io/recipes/nft-portfolio" rel="noreferrer"><strong><u>Build an NFT Portfolio Viewer</u></strong></a>. Holdings grouped by collection with floor prices and chain breakdown.</li><li><a href="https://developers.zerion.io/recipes/ai-agent-integration" rel="noreferrer"><strong><u>Build an AI Agent with Onchain Data</u></strong></a>. Wire Zerion API tools into an agent loop with examples for OpenAI, Anthropic, LangChain, and Vercel AI SDK.</li></ul><p>Each recipe starts with what you're building, lists prerequisites, then walks through the implementation. There is no filler, just the code and context you need.</p><h2 id="quickstart-first-request-in-under-5-minutes">Quickstart: first request in under 5 minutes</h2><p>The new quickstart page replaces the old "Getting Started" section with a guided flow.&nbsp;</p><p>It walks through the steps:&nbsp;</p><ol><li><a href="https://dashboard.zerion.io/" rel="noreferrer">Get your API key</a> from the dashboard,&nbsp;</li><li>Test a request directly in the docs using the built-in playground,&nbsp;</li><li>Make a request from your own code (with examples in cURL, JavaScript, Python, and Go)</li><li>Understand the JSON response format.</li></ol><p>The quickstart also links out to popular entry points (wallet data, transactions, DeFi positions) so you can jump straight to the endpoint you need.</p><h2 id="core-concepts-the-pages-that-were-missing">Core concepts: the pages that were missing</h2><p>Several topics that previously required digging through FAQs or endpoint descriptions now have dedicated pages:</p><ul><li><a href="https://developers.zerion.io/pagination-and-filtering" rel="noreferrer"><strong>Pagination &amp; Filtering</strong></a>. Pagination explained with iteration examples, plus the <code>filter[field]</code> syntax and sorting options.</li><li><a href="https://developers.zerion.io/rate-limits" rel="noreferrer"><strong>Rate Limits</strong></a>. Response headers broken down, retry logic with code examples, and tips for staying under your quota.</li><li><a href="https://developers.zerion.io/webhooks" rel="noreferrer"><strong>Webhooks</strong></a>. The full lifecycle from subscription creation to payload handling, including signature verification, retry behavior, rollback handling, and delivery guarantees.</li><li><a href="https://developers.zerion.io/pricing" rel="noreferrer"><strong>Pricing</strong></a>. Plan details and cost estimation in one place.</li><li><a href="https://developers.zerion.io/error-handling" rel="noreferrer"><strong>Error Handling</strong></a>. Error codes and response formats with guidance on how to handle each.</li></ul><p>These pages make it possible to understand the API's behavior without trial and error.</p><h2 id="build-with-ai-cli-x402-and-mcp">Build with AI: CLI, x402, and MCP</h2><p>The old docs had a brief "Build with AI" page. The new docs expand this into a full section with three integration paths:</p><p><a href="https://developers.zerion.io/build-with-ai/zerion-cli" rel="noreferrer"><strong>Zerion CLI</strong></a> is an open-source command-line tool that wraps the API. Any AI agent that can run shell commands can query wallets, tokens, and chains without writing HTTP integration code.&nbsp;</p><p><a href="https://developers.zerion.io/build-with-ai/x402" rel="noreferrer"><strong>x402 Payments</strong></a> let AI agents pay for API access per-request using USDC on Base, no API key or subscription needed. The docs explain the payment handshake flow and link to the Coinbase x402 SDK for Go, TypeScript, and Python.</p><p><a href="https://developers.zerion.io/build-with-ai/mcp" rel="noreferrer"><strong>MCP Server</strong></a> connects AI tools like Claude Code, Cursor, and VS Code to the Zerion docs and OpenAPI spec via Model Context Protocol. Once connected, your AI assistant can search the full documentation while generating code. The docs include setup instructions for each tool.</p><h2 id="try-it">Try it</h2><p>If you're building with the Zerion API, start with the <a href="https://developers.zerion.io/quickstart" rel="noreferrer"><u>quickstart</u></a> or jump straight into a <a href="https://developers.zerion.io/recipes" rel="noreferrer"><u>recipe</u></a>.</p><p>Create free API keys from the<a href="https://dashboard.zerion.io/"> <u>Zerion Dashboard</u></a>.</p>
