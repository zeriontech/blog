<p>During the NFT mania of 2021, one major criticism was that artists spent a lot of money on gas fees to create NFTs. Now, L2s like Base and Zora drastically reduced costs. But it’s still not free. With <a href="https://zero.network/" rel="noreferrer">ZERϴ Network</a>, you can create NFTs without paying for anything for gas —&nbsp;zero! This post shows how to do this on Highlight.&nbsp;</p><h2 id="what-is-highlight">What is Highlight?&nbsp;</h2><p>Launched in 2022, <a href="https://highlight.xyz/zero"><u>Highlight</u></a> is an NFT platform for creating and collecting onchain art and culture.</p><p>Creating NFTs with Highlight can be done in a few clicks. This can range from simple 1-1 to limited editions and open editions with unlimited mints. For more advanced onchain creators, Highlight offers tools to build interactive generative NFT experiences.&nbsp;</p><p>Highlight is live on many Layer-2 networks, and it’s the first NFT platform to deploy on <a href="https://zero.network/" rel="noreferrer">ZERϴ Network</a>.&nbsp;</p><h2 id="why-zer%CE%B8">Why ZERϴ?&nbsp;</h2><p>The beauty of ZERϴ Network is that transactions are not just cheap. They are free!</p><p>This is not only about costs but also the convenience. New creators can start onchain without the hassle of first bridging to a new network.&nbsp;</p><p>All needed is a <a href="https://link.zerion.io/FPZoKNQywOb" rel="noreferrer">Zerion Wallet</a> and some funds on any chain. Having assets in a wallet is the easiest way to prove you’re not a bot and get free transactions on ZERϴ.&nbsp;You can also <a href="https://help.zerion.io/en/articles/8186414-how-to-import-an-existing-wallet" rel="noreferrer">import your existing wallet into Zerion</a>.</p><p>Here is what you do next.&nbsp;</p><h2 id="steps-to-create-nfts">Steps to Create NFTs&nbsp;</h2><ol><li>Go to <a href="https://highlight.xyz/zero" rel="noreferrer">highlight.xyz/zero</a> and connect your <a href="https://link.zerion.io/FPZoKNQywOb" rel="noreferrer">Zerion Wallet</a></li><li>Click ‘Create’ and select ‘Edition’ — that’s the most simple kind of NFT</li></ol><figure class="kg-card kg-image-card kg-card-hascaption"><img src="https://zerion.io/blog/content/images/2024/11/screely-1731606289975.png" class="kg-image" alt="" loading="lazy" width="2000" height="803" srcset="https://zerion.io/blog/content/images/size/w600/2024/11/screely-1731606289975.png 600w, https://zerion.io/blog/content/images/size/w1000/2024/11/screely-1731606289975.png 1000w, https://zerion.io/blog/content/images/size/w1600/2024/11/screely-1731606289975.png 1600w, https://zerion.io/blog/content/images/size/w2400/2024/11/screely-1731606289975.png 2400w" sizes="(min-width: 720px) 720px"><figcaption><span style="white-space: pre-wrap;">The first step to creating an NFT on Highlight</span></figcaption></figure><ol start="3"><li>Then you need to select ‘Edition type’, here are the options:&nbsp;</li></ol><ul><li><strong>Open</strong> — that’s when you create an NFT that can be minted an unlimited number of times, which is great for building your audience&nbsp;&nbsp;</li><li><strong>Limited </strong>— only a certain number of NFTs can be minted, you can set any number, creating some scarcity&nbsp;</li><li><strong>One of one</strong> — the NFT can be minted just once, making it as unique as it gets&nbsp;</li></ul><p>If you’re just starting, try creating a Open edition and see how many mints you’ll get.
