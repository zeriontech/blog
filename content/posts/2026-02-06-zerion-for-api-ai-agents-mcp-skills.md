<p>To be useful in crypto, AI agents need to access onchain data. Without seeing balances, transaction history, and DeFi positions, they're working blind.</p><p>Zerion API solves this. Built for human developers, it also gives agents instant access to normalized wallet data across EVM and Solana. This post will show you how to get started.&nbsp;</p><h2 id="what-zerion-api-does">What Zerion API does</h2><p><a href="https://zerion.io/api"><u>Zerion API</u></a> provides comprehensive, interpreted wallet data through a single connector:</p><ul><li><strong>Token balances and prices</strong> across all major EVM chains and Solana</li><li><strong>Transaction history</strong> with clear labels</li><li><strong>DeFi positions</strong> from 8,000+ protocols, including lending, liquidity pools, staking, and more</li><li><strong>NFT holdings</strong> and collections with metadata and floor prices</li><li><strong>PnL</strong> for any address and its token positions&nbsp;</li></ul><p>The data is normalized into a consistent schema. Your agent gets the same structure whether it's querying Ethereum, Base, or Solana.</p><h2 id="why-it-works">Why it works</h2><p>Agents need context to be helpful. When you ask "What's my portfolio worth?" or "Show me my recent trades," the agent needs accurate data fast.</p><p>More importantly, an AI agent needs <em>interpreted data. </em>Dealing with raw blockchain data would waste tokens and slow it down.&nbsp;</p><p>Zerion API takes care of that. All data for positions and transactions are already interpreted and ready-to-be used. Created for consumer interfaces, the same endpoints suit AI agents exceedingly well.&nbsp;</p><h2 id="other-ai-agents-already-use-it">Other AI agents already use it&nbsp;</h2><p>Zerion API already powers onchain data for some of the leading AI agents in crypto.</p><ul><li><a href="https://zerion.io/blog/askgina-ai-wallet-companion-built-with-zerion-api/"><u><strong>AskGina</strong> helps users understand their onchain activity</u></a> through natural language queries. It uses Zerion API to pull transaction history and explain complex DeFi interactions in plain English. Gina has access to all endpoints and selects the most useful ones to answer user prompts.&nbsp;</li><li><a href="https://zerion.io/blog/pigeon-an-ai-wallet-built-with-zerion-api/"><u><strong>Pigeon</strong> is an AI wallet</u></a> that interprets user intent and executes transactions. Zerion API provides the wallet context Pigeon needs to make smart suggestions.</li><li><a href="https://zerion.io/blog/guide-to-heyelsa-what-it-is-how-it-works-and-more/"><u><strong>HeyElsa</strong> offers conversational access</u></a> to onchain crypto. Users ask questions about their portfolio in natural language, and Elsa responds with accurate numbers pulled from Zerion API.<a href="https://zerion.io/blog/guide-to-heyelsa-what-it-is-how-it-works-and-more/">&nbsp;</a></li></ul><h2 id="how-to-get-started">How to get started&nbsp;</h2><p>Start by creating a<a href="https://dashboard.zerion.io/"> <u>free API key</u></a>. The process takes less than a minute. The free plan will give you up to 10 calls per second and 2,000 calls per day.&nbsp;&nbsp;</p><p>Once you have your key, you can integrate Zerion API into your agent platform.</p><p>The easiest way to give Zerion API to agents is via Claude: in Claude Code, Cowork, or open-source <a href="https://openclaw.ai/"><u>OpenClaw</u></a> (fka Clawd and Moltbot). All you need are Zerion’s MCP and Skills.&nbsp;</p><h3 id="model-context-protocol-mcp">Model Context Protocol (MCP)</h3><p>The<a href="https://developers.zerion.io/reference/building-with-ai"> <u>Zerion MCP server</u></a> connects Claude (or other LLM) directly to onchain data.&nbsp;</p><ol><li>Open Claude, Settings → Connectors → Add custom connector</li><li>Click “Add Custom Connector”, enter name and remote RPC URL: <a href="https://developers.zerion.io/mcp"><u>https://developers.zerion.io/mcp</u></a>&nbsp;</li><li>In Advanced settings, leave OAuth field empty&nbsp;</li><li>Set tool permissions to “Always allow.”&nbsp;</li></ol><figure class="kg-card kg-image-card"><img src="https://zerion.io/blog/content/images/2026/02/Screenshot-2026-02-06-at-12.23.16.png" class="kg-image" alt="" loading="lazy" width="1544" height="1092" srcset="https://zerion.io/blog/content/images/size/w600/2026/02/Screenshot-2026-02-06-at-12.23.16.png 600w, https://zerion.io/blog/content/images/size/w1000/2026/02/Screenshot-2026-02-06-at-12.23.16.png 1000w, https://zerion.io/blog/content/images/2026/02/Screenshot-2026-02-06-at-12.23.16.png 1544w" sizes="(min-width: 720px) 720px"></figure><p>That’s it. To try it out without complex configurations, <em>"Use my Zerion API key (give it once in that conversation) to get the portfolio for wallet 0x....”</em>. For more security, you should store your key in an environment variable.&nbsp;</p><p>Your agents will now be able to query wallet information in real-time during conversations. You can make it even easier for them by adding Skills.&nbsp;</p><h3 id="claude-skills">Claude Skills</h3><p>Even without Skills, Claude can access Zerion API. After adding the Skill, Claude will now use it to answer prompts. </p><p>With MCP + Skill, Claude can even create useful artefacts that could help you with onchain analysis. </p><p>For example, here's a profile for a wallet address based on a simple prompt: </p><div class="kg-card kg-callout-card kg-callout-card-blue"><div class="kg-callout-emoji">💬</div><div class="kg-callout-text">Use Zerion API MCP to tell me what kind of wallet address &lt;ADDRESS&gt; - I saw it trading a popular coin and what to understand who they are. Use my Zerion API key &lt;KEY&gt; (never share it!)</div></div><figure class="kg-card kg-image-card kg-width-wide"><img src="https://zerion.io/blog/content/images/2026/02/Screenshot-2026-02-06-at-17.17.04.png" class="kg-image" alt="Claude's analysis of a wallet address with Zerion API" loading="lazy" width="1017" height="708" srcset="https://zerion.io/blog/content/images/size/w600/2026/02/Screenshot-2026-02-06-at-17.17.04.png 600w, https://zerion.io/blog/content/images/size/w1000/2026/02/Screenshot-2026-02-06-at-17.17.04.png 1000w, https://zerion.io/blog/content/images/2026/02/Screenshot-2026-02-06-at-17.17.04.png 1017w"></figure><p>You can also teach Claude by giving <a href="https://developers.zerion.io/" rel="noreferrer">links to Zerion API docs</a> and using Claude’s skill-building Skill. It's a great way to create a skill for your unique use cases.</p><h2 id="what%E2%80%99s-next">What’s next&nbsp;</h2><p>This is just the beginning.&nbsp;</p><p>We’re building Zerion API with AI agents in mind as first-class citizens.
