<p>Fantom is a fast and efficient Layer-1 network for smart contracts.</p><p>FTM, its native <a href="https://zerion.io/blog/crypto-coins-vs-tokens-the-difference-explained/">coin</a>, made a spectacular 10x run in 2021 and showed quite a bit of volatility afterward. But many people don’t know what exactly makes Fantom different and why it’s so attractive to app developers.</p><p>In this post we’ll deep dive into the Fantom ecosystem, outlining its history and technology, as well as showcasing top <a href="https://zerion.io/blog/how-to-invest-in-defi/">DeFi</a> applications that you can use with <a href="https://zerion.io/?utm_medium=blog&amp;utm_source=article&amp;utm_campaign=fantom">Zerion Wallet</a>.</p><h2 id="what-is-fantom">What is Fantom?</h2><p>Fantom is a Layer-1 network built using smart contracts that are compatible with Ethereum but process transactions cheaper and faster.</p><p>Unlike some Ethereum alternatives, however, Fantom didn’t sacrifice decentralization to make the network faster.</p><p>We’ll dig into Fantom’s tech deeper but first, let’s see how the Fantom ecosystem developed.</p><h3 id="fantom-history">Fantom history</h3><p>Founded in 2018, the Fantom Foundation was created to solve the blockchain trilemma: ensure decentralization, security, and scalability.</p><p>In summer 2018, Fantom completed a token sale, raising $40 million. Those tokens were subject to vesting, which is now completed and reflected in the circulating supply.</p><p>In December 2019, Fantom launched its mainnet called Opera Chain. It offered smart contracts with full compatibility with Ethereum.</p><p>Fantom’s usage exploded in 2021 when major Ethereum protocols launched versions on the FTM mainnet. The incomplete list includes Curve (February), Sushiswap (March), and Yearn (October).</p><figure class="kg-card kg-image-card kg-card-hascaption"><img src="https://zerion.io/blog/content/images/2022/06/--------------2022-05-31---14.25.52.png" class="kg-image" alt="Transactions on Fantom" loading="lazy" width="2000" height="859" srcset="https://zerion.io/blog/content/images/size/w600/2022/06/--------------2022-05-31---14.25.52.png 600w, https://zerion.io/blog/content/images/size/w1000/2022/06/--------------2022-05-31---14.25.52.png 1000w, https://zerion.io/blog/content/images/size/w1600/2022/06/--------------2022-05-31---14.25.52.png 1600w, https://zerion.io/blog/content/images/size/w2400/2022/06/--------------2022-05-31---14.25.52.png 2400w" sizes="(min-width: 720px) 720px"><figcaption>Transactions on Fantom exploded in 2021</figcaption></figure><p>One of the reasons behind this development and user adoption is the technology Fantom has under the hood.</p><h3 id="fantom%E2%80%99s-technology">Fantom’s technology</h3><p>Fantom has two layers: a consensus mechanism called Lachesis and an execution layer called Opera (aka the mainnet).</p><p><strong>Lachesis</strong> uses a version of a Proof of Stake algorithm that achieves asynchronous Byzantine fault tolerance (aBFT). Asynchronous means that nodes can process data at different times. This enables higher throughput and faster transactions. And Byzantine fault tolerance means that the network can agree on transactions even if some nodes are malicious or fraudulent.</p><p>Fantom is an open network and anyone with 500,000 <a href="https://app.zerion.io/explore/asset/ftm-0x4e15361fd6b4bb609fa63c81a2be19d873717870?utm_medium=blog&amp;utm_source=article&amp;utm_campaign=fantom&amp;utm_content=ftm">FTM</a> can stake their tokens to become a validator. Anyone with over 1 FTM can delegate their tokens and earn staking rewards.</p><p>On top of Lachesis is <strong>Opera</strong>, a smart contracts network compatible with Ethereum Virtual Machine (EVM).</p><p>EVM compatibility means that developers can easily port their Ethereum smart contracts to Fantom and use tools they are familiar with. For developers used to working with Ethereum, this makes development faster than on non-EVM networks like Solana.</p><p>Meanwhile, users can add Fantom as another network in wallets like MetaMask. Smart wallets like <a href="https://link.zerion.io/zWLUQS4avqb">Zerion Wallet</a> already have built-in Fantom support and switching to it is as easy as tapping a toggle.</p><h2 id="what-makes-fantom-different">What makes Fantom different?</h2><p>To summarize, Fantom offers:</p><ul><li><strong>Fast transaction:</strong> 1 second on average</li><li><strong>Low gas fees:</strong> A transaction often costs &gt;$0.01</li><li><strong>EVM compatibility:</strong> Ethereum developers can easy to move apps to Fantom and use familiar tools</li><li><strong>Cosmos SDK compatibility: </strong>Developers can also use Fantom’s Lachesis as a consensus layer and build a new network with Cosmos SDK.</li></ul><h2 id="ftm-vs-other-cryptocurrencies">FTM vs other cryptocurrencies</h2><p><a href="https://app.zerion.io/explore/asset/ftm-0x4e15361fd6b4bb609fa63c81a2be19d873717870?utm_medium=blog&amp;utm_source=article&amp;utm_campaign=fantom&amp;utm_content=ftm">FTM</a> is Fantom’s native cryptocurrency.</p><p>Like ETH for Ethereum or BNB for Binance Smart Chain, FTM is used to pay transaction fees and secure the network. Holders of FTM can also vote on key governance decisions.</p><p>There are several versions of FTM:</p><ul><li>Mainnet FTM — this token is used to pay for gas on the Fantom mainnet, also known as Opera Chain or Fantom Opera</li><li>FTM as ERC-20 — a token on the Ethereum network</li><li>FTM as BEP-20 — a token on the BSC network</li></ul><p>Different versions of FTM can be swapped for one another through bridges.</p><p>To use Fantom applications, you will need to hold the mainnet FTM to pay gas fees.
