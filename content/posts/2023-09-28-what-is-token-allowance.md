<p>If you’ve spent time in crypto, you came across token allowance. But do you really understand what it is? If not, don’t worry. In this post, we’ll explore what token allowance is and then show you how to set a custom token allowance with <a href="https://link.zerion.io/2jNcQs1UsDb">Zerion Wallet</a>, revoke it, and more.</p><h2 id="what-is-token-allowance">What is token allowance?</h2><p>Before any Web3 app can use your tokens, you need to grant it your permission to spend them — this is commonly known as ‘token allowance’ or ‘token approval’.</p><p>These terms come from standard functions of any ERC-20 token smart contract. The ‘approve’ function lets you set the spending allowance for some dapp (like <a href="https://app.zerion.io/swap">Zerion’s swap tab</a>).  The ‘allowance’ function returns how much some dapp can spend on your behalf.</p><p>In practice, you can just sign the transactions with your wallet. Only after granting the approval with your wallet, you get to see the total price of the swap. On the Ethereum network, the gas costs of the swap could be higher than what you wanted to swap… at least, that’s what happened during the bull run.</p><p>So to save on gas costs, most dapps request an unlimited allowance — but you can also customize your token allowance.</p><h2 id="how-to-set-a-custom-token-allowance-with-zerion">How to set a custom token allowance with Zerion</h2><p>Most dapps, by default, request <em>unlimited</em> access to a token. This helps to save on gas on additional approvals for subsequent transactions. However...</p><div class="kg-card kg-callout-card kg-callout-card-yellow"><div class="kg-callout-emoji">⚠️</div><div class="kg-callout-text"><strong>Unlimited token approvals are dangerous</strong>: if the smart contract is hacked or the dapp goes rogue, the infinite allowance allows it to use <em>all</em> your holdings of the approved token.</div></div><p>A scam website can also trick you into granting an unlimited allowance — after that, the scammer will steal all of that token.</p><p>Finally, on <a href="https://zerion.io/blog/layer-1-and-layer-2/">Layer-2 networks</a>, it usually doesn’t make sense to penny-pinch approval transactions. Gas is cheap anyway.</p><p>That’s why, in <a href="https://link.zerion.io/2jNcQs1UsDb">Zerion Wallet</a>, you can customize your token allowance for any approval transaction.</p><figure class="kg-card kg-image-card kg-card-hascaption"><img src="https://zerion.io/blog/content/images/2023/09/custom-token-approval.gif" class="kg-image" alt="screenshot of custom token allowance in Zerion Wallet" loading="lazy" width="960" height="658" srcset="https://zerion.io/blog/content/images/size/w600/2023/09/custom-token-approval.gif 600w, https://zerion.io/blog/content/images/2023/09/custom-token-approval.gif 960w" sizes="(min-width: 720px) 720px"><figcaption>Custom token allowance in Zerion Wallet</figcaption></figure><p>Just set the token allowance to only the amount of tokens you want to spend in that specific transaction. Or you can set the allowance to what you feel is safe for that smart contract: higher for something battle-tested decentralized application like Aave and lower for any degen dapp.</p><p>Of course, you can also revoke any token approvals at any time.</p><h2 id="how-to-check-and-revoke-your-token-approval">How to check and revoke your token approval</h2><p>There are several tools for checking your token approvals.</p><p>The easiest way to check and revoke token approvals is with <a href="https://revoke.cash/">Revoke.cash</a>.</p><p>After connecting your wallet, you get a list of token allowances for various smart contracts across dozens of blockchains. Unfortunately, there is currently no easy way to revoke all token approvals in one transaction.</p><p>That’s why it’s better always to customize token approvals and set a reasonable limit that doesn’t expose you to unlimited risk and still lets you save on gas.</p><h2 id="token-permits-approval-and-spending-in-one-transaction">Token permits: approval and spending in one transaction</h2><p>As you’ve seen, approvals can be expensive, especially if you swap small amounts.</p><p>That’s why devs came up with a way to do both the approval and swap in one transaction.</p><p>Some token smart contracts have a newer function: <em>permit.</em></p><p>The permit function lets you set the allowance and spend tokens in one transaction.</p><p>This saves gas on the first swap for that pair. You also get to see the total gas cost of the swap right away.</p><p>The only problem is that not all tokens and not all dapps support this function.</p><p>The best-known DEX that uses this feature is 1inch. They also have an up-to-date<a href="https://belactriple9.github.io/1inchSupportedPermits/"> list of tokens with permits</a> — it currently includes USDC, DAI, UNI tokens, and other popular tokens.</p><h2 id="conclusion">Conclusion</h2><p>In this post, we explored what token approval and token allowance are, examined why you should set a custom allowance, and how to do this with Zerion Wallet. We then explained how to check and revoke your token approvals. Finally, we explored the newer function of token permits that can help to save on gas by combining approval and swap in one transaction.</p><p>Unlimited token approvals expose you to an unnecessary risk. It’s a good practice to set a custom token allowance, which you can easily do with <a href="https://link.zerion.io/2jNcQs1UsDb">Zerion Wallet</a>.</p><hr><h2 id="faq">FAQ</h2><h3 id="why-is-token-approval-needed">Why is token approval needed?</h3><p>Token approval is a smart contract function that gives a dapp your permission to spend tokens on your behalf. It’s a required transaction that you need to sign before you can do an actual swap or any other transaction with the token.</p><h3 id="what-is-the-difference-between-token-approval-and-allowance">What is the difference between token approval and allowance?</h3><p>Both token approval and allowance are standard functions of an ERC-20 token smart contract.</p><p>Token approval grants the user permission to spend the token. For example, you can approve Uniswap's smart contract to access USDC. This is a required step before you can swap USDC to another token.</p><p>Meanwhile, token allowance returns how much can spent by a specific address — initially, it’s zero, and after a token approval, it’s whatever is set.</p><h3 id="is-there-a-token-allowance-checker">Is there a token allowance checker?</h3><p>​​Yes, with <a href="https://revoke.cash/">Revoke.cash</a>, you can check your allowance across many networks and revoke access. </p><p>You can also check allowance on the Ethereum network with <a href="https://etherscan.io/tokenapprovalchecker">Etherescan’s checker</a>. There are similar features on many blockchain explorers on other networks.</p><figure class="kg-card kg-image-card kg-card-hascaption"><img src="https://zerion.io/blog/content/images/2023/09/Screenshot-2023-09-27-at-15.09.40.png" class="kg-image" alt="screenshot of checking token approvals with Etherscan" loading="lazy" width="2000" height="1010" srcset="https://zerion.io/blog/content/images/size/w600/2023/09/Screenshot-2023-09-27-at-15.09.40.png 600w, https://zerion.io/blog/content/images/size/w1000/2023/09/Screenshot-2023-09-27-at-15.09.40.png 1000w, https://zerion.io/blog/content/images/size/w1600/2023/09/Screenshot-2023-09-27-at-15.09.40.png 1600w, https://zerion.io/blog/content/images/size/w2400/2023/09/Screenshot-2023-09-27-at-15.09.40.png 2400w" sizes="(min-width: 720px) 720px"><figcaption>Checking token approval with Etherscan</figcaption></figure><h3 id="what-does-revoking-the-token-allowance-mean">What does revoking the token allowance mean?</h3><p>Revoking the allowance means you set token approval to zero. After that, the smart contract can no longer spend tokens on your behalf.</p><h3 id="do-nfts-have-token-approvals">Do NFTs have token approvals?</h3><p>Yes, NFT smart contracts also have setApprovalForAll() function that lets a dapp spend all NFTs within that collection. For example, OpenSea can request that approval.</p><h3 id="what-are-the-allowances-in-erc-20">What are the allowances in ERC-20?</h3><p>Upon the approval of the ERC-20 token, the allowance function shows the maximum possible withdrawal by the authorized address.</p><h3 id="why-are-token-allowances-risky">Why are token allowances risky?</h3><p>Token allowances let smart contracts spend tokens on your behalf. Of course, normally, you still initiate any operation with these tokens. But if the smart contract is hacked, attackers may be able to steal all your tokens. With <a href="http://zerion.io/">Zerion Wallet</a>, you can always set custom token allowance. </p>
