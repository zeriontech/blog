<p>We at Zerion are quite opinionated about what a wallet should look like.&nbsp;</p><p>But if, instead of an app, a wallet could exist as just an AI agent with whom you can chat through any messenger?&nbsp;</p><p>That’s what Pigeon set out to build and even more. And to build this ambitious vision, the team used <a href="https://zerion.io/api" rel="noreferrer">Zerion API</a>. Let’s dive in.&nbsp;</p><h2 id="what-is-pigeon">What is Pigeon?&nbsp;</h2><p><a href="https://pigeon.trade/"><u>Pigeon</u></a> is an AI agent wallet that lets users trade using natural language across messengers such as Telegram, Discord, Farcaster, and even Slack. Think of it as Replit or Cursor, but for trading. There's no traditional UI to navigate, just conversations with an AI that understands both coding and crypto.</p><p>The wallet is fully programmable, meaning users can build custom automations directly through chat. Want to track a friend's wallet and automatically copy their trades? Just tell Pigeon what you want, and it handles the rest.</p><p>Pigeon creates wallets automatically on both EVM chains and Solana, supporting all major protocols, including <a href="https://zerion.io/blog/polymarket-how-to-make-track-and-profit-from-predictions/" rel="noreferrer">Polymarket</a>. The AI agent can explain crypto concepts at any knowledge level and works in all languages, making onchain trading accessible to anyone, anywhere.</p><p>The name itself carries meaning.&nbsp;</p><figure class="kg-card kg-embed-card"><blockquote class="twitter-tweet"><p lang="en" dir="ltr">Breaking news: <a href="https://t.co/n90JAraHCi">pic.twitter.com/n90JAraHCi</a></p>— pigeon.trade (@pigeon_trade) <a href="https://twitter.com/pigeon_trade/status/1998421056386961618?ref_src=twsrc%5Etfw">December 9, 2025</a></blockquote>
<script async="" src="https://platform.twitter.com/widgets.js" charset="utf-8"></script></figure><p>Pigeons were historically used as a communication layer for banks, carrying messages between financial institutions. Now, Pigeon serves as a modern communication layer between users and crypto protocols.</p><p>And this requires reliable access to onchain wallet data.&nbsp;</p><h2 id="the-%D1%81hallenge-real-time-position-tracking-across-chains">The сhallenge: Real-time position tracking across chains</h2><p>Like any wallet, Pigeon needs to track what each user’s address holds.&nbsp;</p><p>Every time a user makes a trade through Pigeon, they need to see their updated positions immediately. When they're building automations that track other wallets or copy trades, those positions need to be accurate across all supported chains.</p><p>Pigeon's team tested multiple API providers to find one that could deliver:</p><ul><li><strong>Reliable, consistent responses</strong> across all major chains without unexpected downtime or errors</li><li><strong>Fast team support</strong> when issues arose or new features were needed</li><li><strong>Simple integration</strong> that wouldn't slow down product development</li></ul><p>The decision ultimately came down to developer experience.&nbsp;</p><p>Some providers never responded to inquiries. Others had unreliable endpoints or complicated setup processes. Pigeon needed an API that would work as smoothly as their conversational interface, and they found strong recommendations from other builders pointing to Zerion.</p><h2 id="the-solution-zerion-apis-positions">The solution: Zerion API's positions</h2><p>After evaluating several options, Pigeon integrated Zerion API's <a href="https://developers.zerion.io/reference/listwalletpositions" rel="noreferrer">positions endpoint</a> into their core trading flow.</p><p>The integration was straightforward. Every time a user executes a trade through Pigeon's AI interface, the wallet queries Zerion API to fetch updated position data across all chains. The same endpoints power the custom automations users build, like wallet tracking and trade copying features.</p><p>For Pigeon's architecture, this meant:</p><ul><li><strong>Single source of truth:</strong> One API call retrieves positions across all supported EVM chains and Solana</li><li><strong>Real-time updates:</strong> Position data refreshes after every trade, keeping users informed</li><li><strong>Automation-ready:</strong> Users can leverage Pigeon's Zerion API key to build their own tracking tools</li></ul><p>What sealed the decision was the combination of technical reliability and team responsiveness. While other providers had slow or nonexistent support, Zerion's team responded quickly and made the setup process smooth.</p><h2 id="the-impact-fast-positions-that-just-work">The impact: Fast positions that just work</h2><p>With Zerion API powering position tracking, Pigeon delivers an AI trading experience that works seamlessly across platforms and chains.</p><p>For users, the integration is invisible but essential. Whether they're trading via Telegram, Discord, or Slack, they get instant visibility into their holdings after every transaction. The AI can explain the user’s portfolio in any language, at any technical level, with accurate data underneath.</p><p>For the Pigeon team, the benefits included:</p><ul><li><strong>Development velocity:</strong> Quick integration meant more time building AI features and less time debugging data infrastructure</li><li><strong>Cross-chain coverage:</strong> Support for all major chains and protocols without juggling multiple API providers</li><li><strong>Reliable foundation:</strong> Consistent API responses meant fewer support issues and better user experience</li></ul><blockquote class="kg-blockquote-alt">As <a href="https://www.linkedin.com/in/jasonkimbc/" rel="noreferrer">Jason Kim</a>, founder of Pigeon, put it: <em>"We found Zerion through builder recommendations after testing multiple APIs. Reliable responses and quick team support made the choice clear. Other providers either didn't respond or had inconsistent endpoints. Zerion just worked."</em></blockquote><h2 id="whats-next">What's next</h2><p>Pigeon is expanding their use of Zerion API to include <a href="https://zerion.io/blog/onchain-pnl-api-how-to-track-profit-and-loss-for-wallets-and-tokens/"><u>P&amp;L tracking</u></a> and the <a href="https://developers.zerion.io/reference/listwallettransactions"><u>transaction endpoint</u></a> to add more context.
