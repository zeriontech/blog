<p>Solana is known for speed. Now, with Zerion API, you can build on Solana even faster by accessing onchain data without setting up your own infrastructure. And you can start for free by getting API credits for helping us stress-test the Zerion API on Solana.</p><h2 id="why-zerion-api-on-solana">Why Zerion API on Solana</h2><p>Zerion API makes it easy to build apps that need wallet and portfolio data. Instead of indexing blockchain data yourself, you can fetch everything you need through a single endpoint. It’s the same data that powers Zerion’s portfolio tracker and wallet. Leading crypto teams now use Zerion API to add onchain data: Uniswap, Kraken, Opensea, and many more.</p><p>While Zerion API was originally for EVM chains, now it’s also available on Solana, where you can fetch:&nbsp;</p><ul><li><a href="https://developers.zerion.io/reference/listwalletpositions#/"><strong><u>Wallet positions</u></strong></a> — list of all token and <a href="https://zerion.io/blog/solana-stablecoins-the-complete-guide/" rel="noreferrer">stablecoin</a> holdings for any Solana address</li><li><a href="https://developers.zerion.io/reference/getwalletportfolio"><strong><u>Wallet portfolio</u></strong></a> — values of all token holdings for any Solana address, in a currency of your choice&nbsp;</li><li><a href="https://developers.zerion.io/reference/listwallettransactions#/"><strong><u>Transaction history</u></strong></a> — all transactions associated with a Solana address</li></ul><p>With Zerion API you can integrate Solana into your product in the same way you already fetch Ethereum, Base, Arbitrum, and other EVM data. No need to deal with different explorers or data quirks Zerion API gives you one interface.</p><h2 id="free-credits-for-early-access">Free Credits for Early Access</h2><p>To support builders on Solana, we’re offering <strong>free API credits </strong>for early access to <a href="https://zerion.io/solana-api" rel="noreferrer">Zerion API on Solana</a>.
