<p>Building onchain means dealing with dozens of blockchains, thousands of protocols, and the endless list of tokens. Zerion API takes care of all that complexity. But there are still lots of questions. </p><p>Below are answers to the most common questions developers ask when evaluating Zerion API.</p><h2 id="what-chains-do-you-support">What chains do you support?</h2><p>Zerion API supports 38+ blockchain networks, including Major EVM chains, such as:</p><ul><li><a href="https://zerion.io/api/ethereum" rel="noreferrer">Ethereum mainnet</a></li><li><a href="https://zerion.io/api/polygon" rel="noreferrer">Polygon</a></li><li><a href="https://zerion.io/api/arbitrum" rel="noreferrer">Arbitrum</a></li><li><a href="https://zerion.io/api/optimism" rel="noreferrer">Optimism</a></li><li><a href="https://zerion.io/api/base" rel="noreferrer">Base</a></li><li><a href="https://zerion.io/api/binance-smart-chain" rel="noreferrer">Binance Smart Chain (BSC)</a></li><li><a href="https://zerion.io/api/hyperliquid" rel="noreferrer">Hyperliquid</a></li><li>Tempo</li><li>MegaETH</li></ul><p>… and many others. The up-to-date <a href="https://developers.zerion.io/reference/supported-blockchains"><u>list of supported chains is on this page</u></a>.&nbsp;</p><p>Among non-EVM chains, Zerion API supports Solana, using the same endpoints and schema as for EVM chains. This means you won’t need separate handling for Solana and EVM.&nbsp;</p><p>The complete and current list of supported chains is also available via the <a href="https://developers.zerion.io/reference/listchains"><u>chains endpoint</u></a>.</p><h2 id="what-defi-protocols-do-you-support">What DeFi protocols do you support?</h2><p>Zerion API <a href="https://zerion.io/blog/how-to-fetch-multichain-defi-positions-for-wallet-with-zerion-api/" rel="noreferrer">tracks 8,000+ DeFi protocols</a> across all supported chains. This includes:</p><ul><li>Lending platforms (Aave, Compound, Morpho, etc.)</li><li>DEXs and liquidity pools (Uniswap, Curve, Balancer, Aerodrome, etc.)</li><li>Staking contracts (Lido, Binance staked ETH, Rocket Pool, etc)</li><li>Yield aggregators</li><li>Vesting contracts</li></ul><p>Position types tracked include: deposit, loan, locked, staked, reward, investment positions, and wallet (i.e. held directly, not in a protocol).</p><p>The full list of <a href="https://developers.zerion.io/reference/listdapps"><u>supported protocols</u></a> is available via the dapps endpoint.</p><h2 id="can-zerion-api-handle-both-evm-and-solana">Can Zerion API handle both EVM and Solana?</h2><p>Yes. Zerion API provides unified coverage for both EVM chains and Solana through a single API interface.</p><p>This reduces complexity through:</p><ul><li>Unified schema across EVM and Solana, no chain-specific handling required,</li><li>Consistent metadata across all chains,</li><li>One vendor for both ecosystems, eliminating the need for separate integrations.</li></ul><h2 id="what-are-the-rate-limits-for-zerion-api">What are the rate limits for Zerion API?</h2><p>Rate limits vary by plan, which you can choose in <a href="https://dashboard.zerion.io/" rel="noreferrer">Zerion API Dashboard</a>.</p><ul><li>Developer (Free): 10 requests per second (RPS)</li><li>Builder: 50 RPS</li><li>Startup: 300 RPS</li><li>Enterprise: 1,000+ RPS</li></ul><p>Exceeding rate limits returns HTTP 429 status code (too many requests error).</p><h2 id="how-fast-are-response-times">How fast are response times?</h2><p>Zerion API delivers sub-second response times for most queries:</p><ul><li>Portfolio and balance queries: Typically under 1 second</li><li>Cached data: Returns immediately</li><li>API response times: Millisecond-level latency</li><li>Optional <code>sync=true</code> parameter: Triggers real-time refresh (can take up to 30 seconds for full chain sync)</li></ul><p>The API is optimized specifically for wallet UI loading scenarios where a fast initial render is critical. That's the main reason <a href="https://zerion.io/blog/how-bungee-added-token-lists-with-zerion-api/" rel="noreferrer">why Bungee chose Zerion API</a>. </p><h2 id="whats-your-maximum-call-volume-per-month">What's your maximum call volume per month?</h2><p>The maximum number of requests per month depends on your subscription plan in the <a href="https://dashboard.zerion.io/" rel="noreferrer">API Dashboard</a>.</p><ul><li>Developer (Free): 2000 calls per month</li><li>Builder: 250,000 calls per month</li><li>Startup: 1,000,000</li><li>Growth: 2,500,000</li><li>Enterprise: 2,000,000+</li></ul><p>Enterprise customers can scale significantly beyond standard limits. </p><h2 id="how-fresh-is-the-data-in-zerion-api">How fresh is the data in Zerion API?</h2><p>In Zerion API, data updates within milliseconds of new blocks being confirmed:</p><ul><li>Near real-time data across all supported tokens</li><li>Cached responses return instantly; use <code>sync=true</code> for fresh onchain data</li><li>Webhook subscriptions available for real-time transaction monitoring</li></ul><h2 id="what-is-the-pricing-for-zerion-api">What is the pricing for Zerion API?</h2><p>The subscription price depends on the plan in <a href="https://dashboard.zerion.io/" rel="noreferrer">API dashboard</a>.&nbsp;</p><ul><li>Developer: Free forever after adding a card for verification</li><li>Builder: $149/month&nbsp;</li><li>Startup: $499/month&nbsp;</li><li>Growth: $999/month</li><li>Enterprise: Custom pricing agreement&nbsp;</li></ul><p>Subscriptions are paid automatically with a card via Stripe. Enterprise plans also accept crypto payments (USDC/stablecoins).</p><h2 id="do-you-offer-discounts-for-volume">Do you offer discounts for volume?</h2><p>Enterprise pricing is custom and negotiated based on usage. Please contact <a href="mailto:api@zerion.io"><u>api@zerion.io</u></a> to discuss:</p><ul><li>High-volume pricing</li><li>Custom rate limits</li><li>Specific throughput requirements</li><li>Extended SLAs</li></ul><h2 id="how-quickly-can-we-integrate-zerion-api-and-go-live">How quickly can we integrate Zerion API and go live?</h2><p>Integration is designed to be fast:</p><ul><li>A <a href="https://dashboard.zerion.io/" rel="noreferrer">self-service developer dashboard</a> lets you create API keys and start building in minutes</li><li>Free development keys are available immediately</li><li>Interpreted data (portfolio, positions, transactions) reduces integration time compared to building from raw blockchain data</li><li>Enterprise customers receive dedicated support for faster onboarding</li></ul><p><strong>Case study:</strong> <a href="https://zerion.io/blog/how-opensea-shipped-token-trading-3-months-faster-with-zerion-api/"><u>OpenSea shipped token trading features 3 months faster </u></a>using Zerion API's comprehensive token data.</p><p>Typical integration involves:</p><ol><li>Get API key from developers.zerion.io</li><li>Make authenticated requests (HTTP Basic Auth)</li><li>Parse standardized API responses</li></ol><p>Responses typically already include labels, logos, links, and other metadata that you can use directly in apps without additional enrichment.&nbsp;</p><h2 id="do-you-offer-webhooks">Do you offer webhooks?</h2><p>Yes. Zerion API supports webhook-based subscriptions for real-time transaction monitoring.</p><p>Features include:</p><ul><li>Transaction subscriptions with callback URL configuration</li><li>Callback management for webhook endpoints</li><li>Real-time notifications when transactions occur</li></ul><p>Unlike most other providers, Zerion API’s subscriptions listen to new transactions across <a href="https://developers.zerion.io/reference/supported-blockchains" rel="noreferrer">all supported chains</a><em>.</em></p><h2 id="how-is-pl-calculated">How is P&amp;L calculated?</h2><p>Zerion API calculates <a href="https://zerion.io/blog/onchain-pnl-api-how-to-track-profit-and-loss-for-wallets-and-tokens/" rel="noreferrer">onchain PnL</a> using the FIFO (First In, First Out) accounting standard.</p><p>Metrics in the API response include:&nbsp;</p><ul><li><strong>Realized Gain. </strong>Gain (or loss) from sold assets. Earliest purchases matched with earliest sales (FIFO).</li><li><strong>Unrealized Gain. </strong>Potential gain on unsold assets. Current market value minus cost basis (FIFO).</li><li><strong>Net Invested Amount.</strong> Total invested in assets not yet sold (FIFO).</li><li><strong>Total Fees. </strong>Sum of all transaction fees from asset trades.</li><li><strong>Received External. </strong>Cumulative value of assets received from other wallets.</li><li><strong>Sent External. </strong>Cumulative value of assets received from other wallets.</li><li><strong>Sent for NFTs / Received for NFTs. </strong>Separate tracking for amounts sent/received in NFT transactions.</li></ul><p>You can filter <a href="https://developers.zerion.io/reference/getwalletpnl" rel="noreferrer">PnL responses</a>:</p><ul><li>By specific chains</li><li>By fungible asset IDs (max 100)</li><li>By fungible implementations (<code>chain:address</code> pairs, max 100)</li></ul><p>Assets without available prices are automatically excluded from calculations.</p><h2 id="do-you-have-historical-balance-data">Do you have historical balance data?</h2><p>Yes, Zerion API provides historical data through:</p><ul><li><a href="https://developers.zerion.io/reference/getwalletchart" rel="noreferrer">Balance charts</a> showing wallet value over time</li><li><a href="https://developers.zerion.io/reference/listwallettransactions" rel="noreferrer">Transaction history</a> with decoded operations across all supported chains</li></ul><p>The API supports chart data for both individual assets and overall wallet performance, enabling portfolio trackers and tax/reporting tools to pull historical data for analysis.</p><h2 id="try-it-in-action">Try it in action</h2><p>The best way to see if Zerion API is for you is to <a href="https://dashboard.zerion.io/" rel="noreferrer">get a free API key</a> and try it. </p><p>And if you have any other questions, send an email to api@zerion.io. </p>
