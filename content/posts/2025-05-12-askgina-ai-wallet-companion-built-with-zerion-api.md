<p>When AI meets onchain crypto data, new possibilities emerge. That’s precisely what happened at FarHack, the hackathon at FarCon, where AskGina took center stage as an AI agent changing the way we interact with crypto. And behind Gina’s new onchain powers? Zerion API.</p><p>In this case study, we explore how AskGina integrated the Zerion API to give AI access to onchain data.</p><h2 id="what-is-gina">What is Gina?&nbsp;</h2><p><a href="https://askgina.ai/"><u>AskGina</u></a> is an AI wallet companion that helps you navigate crypto.&nbsp;</p><p>Think of it as ChatGPT for onchain finance, but with more context. Gina can answer questions, explain tokens, and uncover insights tailored to your crypto activity.</p><p>Gina started as an AI agent living on <a href="https://zerion.io/blog/how-to-use-farcaster-and-warpcast-zerion-blog/"><u>Farcaster, a decentralized social network</u></a>.&nbsp;</p><h3 id="farcaster-ai-agent">Farcaster AI Agent</h3><p>Launched in August 2024, Gina quickly gained traction. Farcaster users tagged her in questions in their posts or replies, and answered the questions. This is similar to how Grok now works on X. But the great thing about Farcaster is that every user has built-in and connected wallets.</p><p>So users wanted more. They weren’t just asking, "What’s the price of ETH?" They started saying things like:</p><blockquote>"Gina, can I just give you my money to manage?"</blockquote><p>This marked a turning point. People didn’t just want answers. They wanted actions. They wanted an AI that could:</p><ul><li>Check their wallet balances</li><li>Track portfolio performance across wallets&nbsp;</li><li>Buy, sell, rebalance, and take other actions</li></ul><p>Gina needed to level up.</p><h3 id="farhack-gina-gets-even-smarter">FarHack: Gina Gets Even Smarter&nbsp;</h3><p>The team behind Gina used <a href="https://farhack.xyz/"><u>FarHack</u></a>, the hackathon day at <a href="https://farcon.xyz/"><u>FarCon</u></a>, as an opportunity to make Gina smarter.&nbsp;</p><p>The goal was to give Gina the ability to fetch live onchain data and wallet information, all while staying inside a conversational interface.</p><figure class="kg-card kg-image-card kg-card-hascaption"><img src="https://zerion.io/blog/content/images/2025/05/Askgina-screenshot.png" class="kg-image" alt="Screenshot of Ask Gina, AI wallet companion" loading="lazy" width="1595" height="938" srcset="https://zerion.io/blog/content/images/size/w600/2025/05/Askgina-screenshot.png 600w, https://zerion.io/blog/content/images/size/w1000/2025/05/Askgina-screenshot.png 1000w, https://zerion.io/blog/content/images/2025/05/Askgina-screenshot.png 1595w" sizes="(min-width: 720px) 720px"><figcaption><span style="white-space: pre-wrap;">Ask Gina: AI wallet companion</span></figcaption></figure><p>This challenge led them to the <a href="https://zerion.io/api" rel="noreferrer">Zerion API</a>, which offered one of the prizes at FarCon.&nbsp;</p><h2 id="the-problem-accessing-onchain-data">The Problem: Accessing Onchain Data&nbsp;</h2><p>Crypto data is messy.&nbsp;</p><p>Wallets are spread across multiple chains. Tokens are staked or deposited in various DeFi protocols. Raw block data misses metadata and relations between different elements, making it hard to interpret. Stitching everything together requires infrastructure, which takes time to set up.</p><p>The Gina team knew the data problem well. One of the co-founders, <a href="https://warpcast.com/sidshekhar"><u>Sid</u></a>, previously built a crypto data company that was acquired by Coinbase.&nbsp;</p><p>For Gina to work as a true AI wallet companion, it needed:</p><ul><li>Unified access to wallet balances across chains</li><li>Clean, readable transaction histories</li><li>Reliable token metadata</li></ul><p>In short, Gina needed a one-stop shop for onchain data.</p><h2 id="the-solution-ai-agent-selects-zerion-api-endpoints">The Solution: AI Agent Selects Zerion API Endpoints</h2><p>That’s where Zerion API came in.</p><p>Zerion API offers a set of endpoints that provide easy-to-read data about any address’s transactions, tokens, DeFi positions, NFTs, and more. It’s the same data you can see in the Zerion wallet. And it’s also perfect for giving <a href="https://zerion.io/blog/beyond-chatbots-ai-agents-need-onchain-data/"><u>AI agents access to onchain data</u></a>.&nbsp;&nbsp;</p><p>During FarHack, in just 24 hours, the Gina team integrated Zerion API to give it new features:</p><ul><li><strong>Fetch balances and PnL by Farcaster username:</strong> Instead of typing out wallet addresses, users could simply say "Check @username’s wallet" and Gina would query Zerion API to fetch balances.</li><li><strong>Full balance history:</strong> Gina could now show a user’s portfolio performance over time, powered by Zerion’s historical balance endpoints.</li><li><strong>A Gina Mini-app:</strong> Built with Farcaster’s Mini Kit, the team launched a prototype app where users could share their conversations with Gina.</li></ul><p>To do all this, under the hood, Gina’s AI subagents select the Zerion API endpoints that are best suited to answer the user's query: <a href="https://developers.zerion.io/reference/getwalletchart#/"><u>wallet balance</u></a>, <a href="https://developers.zerion.io/reference/listwallettransactions#/"><u>transaction history</u></a>, <a href="https://developers.zerion.io/reference/getwalletpnl#/"><u>wallet PnL</u></a>, and more. For this impressive integration, the team won Zerion’s FarHack bounty!&nbsp;</p><p>Gina’s capabilities go far beyond simply checking balances. The best way to see what she can do is to <a href="https://askgina.ai/"><u>try it yourself</u></a>.</p><h2 id="what%E2%80%99s-next-for-askgina">What’s Next for AskGina</h2><p>The Gina team isn’t stopping here. Next up:</p><ul><li><strong>Better market and token data:</strong> Integrating deeper insights into token fundamentals and price action.</li><li><strong>Automated actions: </strong>Imagine telling Gina, "Monitor what happens with this token and sell if it drops 10%." That’s coming.</li><li><strong>More platforms:</strong> Expanding Gina’s presence beyond Farcaster to Telegram and other social apps.</li></ul><h2 id="why-it-matters">Why It Matters</h2><p>AskGina is part of a growing wave of AI agents that need onchain data to act intelligently.&nbsp;</p><p>By integrating Zerion API, the Gina team saved weeks of work they would have spent building infrastructure from scratch.</p><p>For developers looking to build crypto-native AI products, Gina’s story is proof: Zerion API unlocks onchain intelligence, fast.</p><p>Ready to give your AI app onchain superpowers? <a href="https://dashboard.zerion.io/?utm_source=blog&amp;utm_medium=article&amp;utm_campaign=askgina-case-study" rel="noreferrer">Get your Zerion API dev key</a>.</p>
