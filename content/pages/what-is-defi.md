<p><em>Understand what decentralized finance (DeFi) is and learn what differentiates it from centralized finance (CeFi)</em></p><p>DeFi is short for decentralized finance, which is an umbrella term for services like investing, borrowing, lending, and trading based on decentralized, non-custodial infrastructure commonly built on the Ethereum blockchain. DeFi’s opposite is CeFi – centralized finance – such as traditional banks and most cryptocurrency exchanges.</p><p>Here’s a list of just a few things that differentiate DeFi from CeFi:</p><ul><li>DeFi is <strong>programmable</strong>, which means that transactions are controlled by smart contracts and not people. In theory, dApps (decentralized apps) can run themselves with virtually no human intervention.</li><li>DeFi is <strong>transparent</strong>. Anyone can access the code to a smart contract in order to understand how it functions or search for bugs. All transaction activity is also publically viewable on sites like <a href="http://www.etherscan.io/" rel="nofollow noopener noreferrer">etherscan.io</a>.</li><li>DeFi is <strong>borderless</strong>. Because dApps do not have any form of KYC (Know Your Customer) processes, they are immediately accessible to anyone with an internet connection, a smart phone, and an <a href="https://zerion.io/ethereum-wallet" rel="noreferrer">Ethereum wallet</a>. This is why Zerion does not ask for user emails and does not have user accounts.</li><li>DeFi is <strong>permissionless</strong>, which means there are no gatekeepers restricting who can create and participate in DeFi services.</li><li>DeFi is <strong>trustless</strong> and <strong>non-custodial</strong>, which means there are no middlemen. Users interact directly with smart contracts and are therefore the sole custodians of their assets at all times.</li><li>DeFi is <strong>interoperable</strong>. Different applications can be built on top of each other and customized in new ways, regardless of who created them. The entire system is a bit like a box of Lego blocks – there is no limit to what can be built.</li><li>DeFi enables a <strong>customizable user experience</strong>. Users aren’t restricted to using only one interface. Smart contracts are like APIs that can be accessed through any platform.</li></ul>
