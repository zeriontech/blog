<p>Zerion Wallet is a smart wallet and the only app you need to manage your whole DeFi and NFT portfolio.</p><p>With Zerion, you can store and track all your assets across 11+ networks and sign any dapp transaction on your mobile and desktop.</p><p>In this post, we’ll show you how to create your first Zerion Wallet and set it up.</p><h2 id="step-1-install-zerion-wallet-mobile-app">Step 1: Install Zerion Wallet mobile app</h2><p>The first step is easy: download the app from <a href="https://link.zerion.io/ZxqPcKvcRtb">Apple App Store</a> or <a href="https://link.zerion.io/ZxqPcKvcRtb">Google Play</a> or install an <a href="https://s3.amazonaws.com/android.utils/apks/apk_29.09.2022.15.23_2.8.5_208503-bitrise-signed.apk">Android APK</a>.</p><h2 id="step-2-create-a-new-zerion-wallet">Step 2: Create a new Zerion wallet </h2><p>On the first screen you’ll see several options:</p><ul><li><strong>Create new wallet </strong>— set up a brand new address</li><li><strong>Import existing wallet</strong> — you can import MetaMask, Trust, or any other wallet you already have via its Secret Recovery Phrase (aka seed phrase, recovery phrase, or secret phrase) or the private key. </li><li><strong>Connect Ledger</strong> —&nbsp;you should not export seed phrase or private key from your hardware wallet. Instead, you can connect your <a href="https://www.ledger.com/" rel="noreferrer">Ledger crypto wallet</a> and swap, bridge, and sign any transaction with Zerion.</li><li><strong>Connect any wallet</strong> — you can easily add any other support mobile wallet you have on your phone and Zerion will track as part of your portfolio</li></ul><p>The easiest way to just create a new <a href="https://link.zerion.io/ZxqPcKvcRtb">Zerion wallet</a>: just select 'Create new wallet' and follow the instructions to create an address. </p><figure class="kg-card kg-image-card kg-card-hascaption"><img src="https://zerion.io/blog/content/images/2022/10/create-Zerion-Wallet.png" class="kg-image" alt="Create new Zerion Wallet" loading="lazy" width="1280" height="1280" srcset="https://zerion.io/blog/content/images/size/w600/2022/10/create-Zerion-Wallet.png 600w, https://zerion.io/blog/content/images/size/w1000/2022/10/create-Zerion-Wallet.png 1000w, https://zerion.io/blog/content/images/2022/10/create-Zerion-Wallet.png 1280w" sizes="(min-width: 720px) 720px"><figcaption><span style="white-space: pre-wrap;">Create new Zerion Wallet</span></figcaption></figure><p>You can use the same address for Ethereum, Polygon, Binance Smart Chain, and all other networks that Zerion supports.</p><h2 id="step-3-backup-your-recovery-phrase">Step 3: Backup your recovery phrase</h2><p>After you create your new address, you need to save your recovery phrase.</p><p>You don't need to do this immediately, it's ok to save it later. You’ll see a notification in Settings with a message about the importance of saving your secret recovery phrase.</p><figure class="kg-card kg-image-card kg-card-hascaption"><img src="https://zerion.io/blog/content/images/2022/10/screely-1664908040510.png" class="kg-image" alt="If you don't save your secret recovery phrase, you'll see a notification in Settings" loading="lazy" width="1128" height="234" srcset="https://zerion.io/blog/content/images/size/w600/2022/10/screely-1664908040510.png 600w, https://zerion.io/blog/content/images/size/w1000/2022/10/screely-1664908040510.png 1000w, https://zerion.io/blog/content/images/2022/10/screely-1664908040510.png 1128w" sizes="(min-width: 720px) 720px"><figcaption><span style="white-space: pre-wrap;">If you don't save your secret recovery phrase, you'll see a notification in Settings</span></figcaption></figure><p>You can create additional addresses in the same wallet group (aka backup group) without saving a new secret recovery phrase. All addresses in a group come from the same recovery phrase. Yet nobody (not even Zerion) can guess that these addresses are from the same group.</p><h2 id="step-4-track-any-wallet">Step 4: Track any wallet</h2><p>Your newly created address is empty and it address</a>:</p><ol><li>From the home screen in<a href="https://link.zerion.io/ZxqPcKvcRtb"> Zerion Wallet app</a>, tap on the wallet icon in the upper right corner</li><li>Select ‘track any wallets</h2><p>Now that you can see how Zerion tracks a portfolio, you might want to move your other wallets to Zerion.</p><p>That way, you’ll be able to track and manage your entire portfolio from one app. You’ll be able to also sign any transactions on the go and trade at the best rates using <a href="https://zerion.io/blog/how-does-trading-aggregation-work-on-zerion/">Zerion’s swap</a>
